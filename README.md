# Discord utility bot

A discord bot written in python, intended for managing smaller discord servers.

[Invite link](https://discord.com/oauth2/authorize?client_id=763366736586080257&scope=bot&permissions=268512256)

## features

- Adding and removing roles with button clicks,
- hosting polls,
- clearing chat,
- hosting games:
  - rock-paper-scissors,
  - connect-four,
  - hangman
- Customizable:
  - prefix,
  - welcome text,
  - which roles are allowed to use a command,
  - time before the auto-deleting messages are deleted

For more information type `-help` in discord channel, after adding the bot.

## required permissions

- necessary:

  - Read Messages
  - Send Messages
  - Read Message History

- optional:
  - Manage Roles (adding and removing roles with buttons)
  - Manage Messages (clearing chat)
