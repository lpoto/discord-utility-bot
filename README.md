# Discord utility bot
A discord bot written in python, intended for managing smaller discord servers.

[Invite link](https://discord.com/oauth2/authorize?client_id=763366736586080257&scope=bot&permissions=268528720)

## features

* Adding and removing roles with reactions,
* hosting polls,
* clearing chat,
* providing server and user info,
* Customizable: 
	- prefix,
	- role managing channel,
	- which roles can use a command,
	- optional welcome text

For more information type `-help` in discord channel, after adding the bot.
