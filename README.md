# Discord utility bot
A discord bot written in python, intended for managing smaller discord servers.

[Invite link](https://discord.com/oauth2/authorize?client_id=763366736586080257&scope=bot&permissions=1342782672)

## features

* Adding and removing roles with reactions,
* hosting polls (TODO),
* clearing chat,
* providing server and user info,
* customizable settings
	- see `-config help` in discord channel

For more information type `-help` in discord channel, after adding the bot.
