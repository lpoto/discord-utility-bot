# Discord utility bot
A discord bot written in python, intended for managing smaller discord servers.

[Invite link](https://discord.com/oauth2/authorize?client_id=763366736586080257&scope=bot&permissions=309506149376)

## features

* Adding and removing roles with button clicks,
* hosting polls,
* hosting games:
	- hangman,
	- rock-paper-scissors,
	- connect-four
* All commands available from the same, easy to use, interface

For more information mention bot in the discord channel

## customizable

* which roles are allowed to use a command

## required permissions

* Read Messages
* Send Messages
* Read Message History
* Manage Roles (adding and removing roles with buttons)
* Create Public Threads
