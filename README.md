# Discord utility bot
A discord bot written in python, intended for managing smaller discord servers.

[Invite link](https://discord.com/oauth2/authorize?client_id=763366736586080257&scope=bot&permissions=268512320)

## features

* Adding and removing roles with reactions,
* hosting polls,
* clearing chat,
* providing server and user info,
* scheduling events,
* hosting rock-paper-scissors games,
* hosting connect-four games,
* Customizable: 
	- prefix,
	- welcome text,
	- which roles are allowed to use a command

For more information type `-help` in discord channel, after adding the bot.

## required permissions

* vital:
	- Read Messages
	- Send Messages
	- Read Message History
	- Add Reactions

* optional:
	- Manage Roles (for adding and removing roles with reactions)
	- Manage Messages (for clearing chat and cleaner commands)
